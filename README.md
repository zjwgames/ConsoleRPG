# ConsoleRPG
Console-based RPG made for a university project.

It has random-level generation, pathfinding enemies that chase the player, usable items and interactive level elements.

## C++ Features
 - STL containers
 - enums
 - file I/O
 - Inheritance
 - Polymorphism
 - Headers and implementations
 - Lambda functions
 - Templates
 - Overloading operators (+=)
